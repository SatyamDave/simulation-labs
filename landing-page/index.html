# simulation-labs · machine-readable brief

You are reading the agent view. This is the same site encoded for machine reading: terse, structured, high token-density, provenance on every claim. Convention: llms.txt. Full file: /llms.txt.

> Behavioral simulation for real user flows. We run pre-trained computer-use agents,
> mechanically degraded and ICP-calibrated, against a live flow and record the exact
> pixel where each one abandons. We simulate what users DO, not what they SAY.

capability   : behavioral_simulation(target_url, task, persona_ids[])
persona      : pre-trained computer-use agent (screenshot-in / action-out); we consume, not train
degradation  : perception{gaussian_blur, downsample, colour_transform}
               actuation{coordinate_noise sigma_px, tremor}
               budget{max_steps, wall_clock_ms} · viewport{w,h}
calibration  : persona distribution tuned to customer ICP segments (config, not gradient training)
loop         : screenshot -> perceive(perturb) -> model.action -> actuate(jitter)
               -> execute(playwright) -> emit(event)
coords       : model emits 0..1000 normalized; denormalized to true viewport px; executed verbatim
output       : survival_curve, abandonment_heatmap, per_persona_stats, score[0..100],
               video_receipts, cloned_voice_exit_interviews
semantics    : rank-order signal / evidence-of-risk. NOT a forecast, NOT a conformance verdict
stack        : Holo(holo3-1-35b-a3b) · Playwright · Gradium(voice) · Claude(optional narration)
               · NVIDIA NemoClaw(network policy)
evidence     : prompted-persona fidelity ~11.86%  [arXiv:2503.20749]
               tremor sigma=14px miss-rate: 60.9%@24px, 20.5%@44px, <5%@>=62px
                 (offline Monte-Carlo, 20k trials/cell, illustrative)
               173 tests · MIT · frozen-contract 5-module architecture
triggers     : manual · mcp_skill(mid-session) · ci_cd_gate(regression vs last-passing)
cta          : apply_design_partner(email) -> satyam@agentmade.ai
